# 📊 AI-Powered Excel Dashboard with VBA Automation

## Overview

An intelligent Excel-based analytics dashboard built using **Excel VBA, Pivot Tables, Slicers, Charts, and AI-assisted development** to automate reporting and generate interactive business insights.

## 🚀 Project Highlights

* Built an interactive dashboard in **Excel (.xlsm)** with VBA automation
* Developed using **AI-assisted logic + VBA scripting** for productivity and smart automation
* Used **Pivot Tables, Pivot Charts, and Slicers** for dynamic analysis
* Created an executive-style dashboard for monitoring key business metrics
* Automated report refresh and user-driven filtering workflows

## 📌 Features

### Interactive Dashboard

* Sales performance monitoring
* Profit analysis tracking
* Quantity and discount insights
* Regional performance breakdown
* Category and sub-category analysis
* Dynamic filtering through slicers

### VBA Automation

* Automated dashboard interactions
* Macro-enabled reporting workflow
* Reduced manual data processing effort
* Repeatable and scalable reporting structure

### Data Analysis Components

The workbook includes:

* **Data Sheet** for raw transactional data
* **Pivot Sheet** for summarized model calculations
* **Dashboard Sheet** for executive reporting visuals

4. Use slicers and controls to interact with the dashboard
5. Run VBA automation buttons (if assigned)
